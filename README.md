### Task 2 – Samajh AI

**Understand government documents in simple language.**

Samajh AI is a privacy-first web application that helps Indian citizens understand complex government documents using on-device AI. Users can upload government documents and receive simplified explanations, important deadlines, action items, and fraud warnings while keeping all data completely private.

**Key Features**

* AI-powered document explanation
* On-device OCR and AI processing
* Offline support
* Fraud detection
* Multi-language support
* Local document history
* Camera document scanning

**Tech Stack**

* React + Vite
* Tailwind CSS
* Tesseract.js
* WebLLM (Gemma 2B)
* IndexedDB
* Framer Motion
* PDF.js
