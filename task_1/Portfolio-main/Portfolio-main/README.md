# Aryan Rane — Portfolio

Personal portfolio site. Built with plain HTML, CSS, and vanilla JS — no framework, no build step.

**Live:** https://dreameterftw.github.io/Portfolio

---

## Stack

- HTML / CSS / Vanilla JS
- Fonts: Instrument Serif, Inter, JetBrains Mono (Google Fonts)
- Hosted on GitHub Pages

## Features

- Black & white editorial design
- Smooth infinite ticker strip
- Typewriter cycling animation on contact heading
- Scroll-reveal animations
- Live Mumbai time in navbar
- Responsive across all screen sizes

## Projects featured

- [Fitry](https://fitry-2df38.web.app/) — Career-first tech learning platform
- [Lex AI](https://lex-ai-gg.web.app/) — Legal document assistant in plain language
- [Cura](https://cura-medical-app.vercel.app/) — Medical symptom companion
- [Nova Health AI](https://nova-health-ai-five.vercel.app/) — AI-assisted personal health tool

## Running locally

No install needed. Just open `index.html` in a browser, or serve with:

```bash
python -m http.server 3000
```

Then visit `http://localhost:3000`.
